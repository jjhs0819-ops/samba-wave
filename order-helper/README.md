# 삼바 주문도우미 (Order Helper) — 독립 확장프로그램

반자동 무신사 주문처리용 **독립 크롬 확장**. 기존 SAMBA-WAVE 확장과 분리되어 있어
원작자 코드 업데이트와 충돌하지 않으며, 추후 삼바 확장으로 이전(이사) 가능하게 모듈화했다.

## 반자동 플로우 (목표)

```
삼바 [원문링크] 클릭
   └→ 주문 데이터 저장 + 무신사 상품탭 오픈      (← 다음 단계에서 삼바 연동)
무신사 (자동)
   └→ 옵션선택 → 구매하기 → 주문서 → 배송지 자동입력 → 멈춤
[사람이 수동] 쿠폰·결제수단 선택 → 결제 → 비번 입력
무신사 주문완료 (자동)
   └→ 주문번호·결제금액 스크랩 → 삼바 자동기입   (← 다음 단계에서 삼바 연동)
```

현재 구현: **무신사 쪽 자동화(주소입력까지) + 결제완료 감지/스크랩**.
삼바 트리거/기입(writeback)은 다음 단계.

## 설치 (맥미니 크롬에서, 무신사 로그인된 그 크롬)

1. 크롬 주소창에 `chrome://extensions` 입력
2. 우측 상단 **개발자 모드** ON
3. **압축해제된 확장 프로그램을 로드** 클릭
4. 이 `order-helper` 폴더 선택
5. 무신사에 로그인되어 있어야 함

## 테스트

1. 확장 아이콘(퍼즐 모양 → 삼바 주문도우미) 클릭 → 팝업
2. 값(상품URL/사이즈/고객정보) 확인 후 **자동주문 시작**
3. 새 탭에서 자동 진행: 옵션선택 → 구매하기 → 주문서 → 배송지 자동입력 → 멈춤
4. 화면 상단 배너로 진행상황 표시
5. (선택) 직접 결제까지 진행하면 완료 페이지에서 주문번호 자동 감지

## 파일 구조

| 파일 | 역할 |
|---|---|
| `manifest.json` | 확장 정의 (MV3) |
| `config.js` | 셀렉터/설정 (무신사 UI 변경 시 여기만 수정) |
| `content-musinsa.js` | 무신사 자동화 상태머신 (옵션/구매/주소/결과) |
| `background.js` | 메시지 허브 + 삼바 writeback 중계 |
| `popup.html` / `popup.js` | 테스트용 작업 시작 트리거 |
