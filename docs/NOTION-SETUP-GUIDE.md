# 노션 워크스페이스 셋업 가이드

> 5분이면 끝남. 아래 순서대로 따라하기

---

## 1단계: 노션 워크스페이스 생성

1. https://notion.so 접속 → 회원가입 (구글 로그인 추천)
2. "팀 워크스페이스 만들기" 선택
3. 이름: `삼바웨이브`
4. 무료 플랜으로 시작 (5명까지 무료)

---

## 2단계: 페이지 구조 만들기

왼쪽 사이드바에서 "새 페이지" 클릭하며 아래 구조를 만든다.

```
삼바웨이브 (최상위 페이지)
├── 대시보드          ← project-dashboard.html 내용 복붙
├── 로드맵            ← ROADMAP.md 내용 복붙
├── 주간회의록
│   ├── 2026-W13 (3/24~3/28)
│   └── (매주 추가)
├── 태스크 보드        ← 데이터베이스 (Board 뷰)
├── 소싱처 현황        ← 데이터베이스 (Table 뷰)
├── 판매처 현황        ← 데이터베이스 (Table 뷰)
├── 개발 가이드
│   ├── 플러그인 스펙   ← PLUGIN-SPEC.md 내용 복붙
│   ├── 팀 규칙        ← TEAM-GUIDE.md 내용 복붙
│   └── 현재 상태      ← CURRENT-STATUS.md 내용 복붙
└── 트러블슈팅 로그     ← 겪은 문제 + 해결 방법 기록
```

---

## 3단계: 태스크 보드 만들기 (칸반)

1. "태스크 보드" 페이지에서 `/board` 입력 → Board 뷰 선택
2. 컬럼 설정:
   - `대기` | `진행 중` | `리뷰 중` | `완료`
3. 속성 추가:
   - **담당자** (Person) — 누가 하는지
   - **우선순위** (Select) — 긴급/높음/보통/낮음
   - **Phase** (Select) — Phase 1/2/3
   - **마감일** (Date)
4. Phase 1 태스크 카드 추가 (위 HTML의 체크리스트 참고)

---

## 4단계: 소싱처/판매처 현황 DB

1. "소싱처 현황" 페이지에서 `/table` 입력 → Table 뷰 선택
2. 컬럼:
   - **소싱처명** (Title)
   - **상태** (Select) — 운영중/개발중/예정
   - **수집** (Checkbox)
   - **갱신** (Checkbox)
   - **담당자** (Person)
   - **비고** (Text)
3. 판매처도 동일하게

---

## 5단계: 팀원 초대

1. 왼쪽 상단 워크스페이스 이름 클릭 → Settings
2. Members → Invite → 이메일 입력
3. 역할: Member (전원 편집 가능)

---

## 팁

### 빠른 입력
- `/h1` → 제목
- `/todo` → 체크리스트
- `/table` → 테이블
- `/board` → 칸반보드
- `/callout` → 강조 박스
- `@이름` → 멘션 (알림 감)

### 매주 하는 것
- 금요일: 주간회의록 페이지 추가
- 월요일: 태스크 보드에서 이번 주 할 일 "진행 중"으로 이동
- 매일: 완료된 태스크 "완료" 컬럼으로 이동

### 안 해도 되는 것
- 예쁘게 꾸미기 (내용이 중요)
- 모든 것 기록 (결정사항 + 블로커만)
- 긴 글 쓰기 (3줄 요약이면 충분)
