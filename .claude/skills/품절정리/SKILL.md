# 품절정리 / 오토튠 점검 절차

## 1. 오토튠(워룸) 상태 점검

- [ ] 각 PC(device)별 갱신 사이클이 정상 동작 중인지 확인 (/warroom)
- [ ] 마지막 갱신 시각(last_refreshed_at)이 24~48시간 이내인지 확인
- [ ] 멈춰있는 PC가 있으면: 로그 확인 → 재시작 → 그래도 안 되면 `/이슈보고` 후보
- [ ] 롯데온 헤드리스 데몬 사용 시: 데몬 버전/동작 상태 확인

## 2. 품절 상품 정리

- [ ] `is_sold_out=true` 인데 아직 마켓에 남아있는 상품 확인
- [ ] `lock_delete=true` 상품은 **절대 삭제하지 않는다** (수동 보호 상품)
- [ ] 삭제 대상 개수를 보고하고 **확인받은 뒤** 마켓 삭제 실행
- [ ] 삭제 후 마켓별 성공/실패 집계

## 3. 고아 상품 정리 (orphan cleanup)

- [ ] 마켓에서는 지워졌는데 삼바 DB에 남아있는 상품 탐지
- [ ] 정리 대상을 보고 후 정리 실행
