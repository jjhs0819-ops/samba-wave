# 상품가공 절차

## 시작 전 확인

1. 가공 대상 확인: 특정 상품? 수집 그룹 전체? 오늘 수집분 전체?
2. 가공 범위 확인: 이미지만 / 상품명만 / 전체

## 절차

### 1. 이미지 가공
- 삼바 이미지 필터(AI) 실행: 모델컷·연출컷·배너 제거, 제품컷만 유지.
- 적용 범위 선택: 대표이미지만 / 상세페이지만 / 전체
- 필터 결과 중 애매한 것(제품컷인지 불확실)은 목록으로 보고하고 사장님 판단을 받는다.

### 2. 상품명/옵션명 정리
- 상품명 규칙(name rules)에 따라 마켓별 상품명 생성.
- 금지어 검사: 정책의 금지어/삭제어 목록 통과 확인. 걸린 단어는 어떤 상품에서 걸렸는지 보고.
- 브랜드명, 모델명이 깨지지 않았는지 샘플 확인.

### 3. 카테고리 매핑
- 소싱처 카테고리 → 마켓 카테고리 매핑 확인.
- 미매핑 상품은 AI 매핑 실행 후, **결과를 표로 보고하고 확인받은 뒤** 확정한다.

### 4. 상세페이지 템플릿
- 상세 템플릿이 적용됐는지 확인. 어떤 템플릿을 쓸지 모호하면 물어본다.

## 가공 후 점검

- [ ] 금지어 0건
- [ ] 카테고리 미매핑 0건
- [ ] 이미지 0장인 상품 없음
- [ ] 가공 완료 수 / 보류(사장님 확인 필요) 수 요약 보고
