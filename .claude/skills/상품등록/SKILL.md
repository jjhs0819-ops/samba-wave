# 상품등록 절차

## 시작 전 확인 (필수)

1. 등록 대상: 어떤 상품들? 몇 개?
2. 대상 마켓과 계정: 어느 마켓, 어느 계정으로?
3. 전송 범위: 신규 등록 / 가격만 / 재고만 / 이미지만 / 상세설명만
4. **실제 전송 전에 위 내용을 요약해서 사장님에게 확인받는다.** (마켓에 올라가는 작업은 되돌리기 어려움)

## 절차

### 1. 등록 전 검증
- [ ] 가공 완료 상태인지 (`/상품가공` 점검 항목 통과)
- [ ] 가격 정책 적용 결과 샘플 확인: 원가 → 판매가 계산이 마진율/최소마진에 맞는지 2~3개 검산
- [ ] 마켓별 필수 항목 누락 없는지 (SSG 고시정보, 원산지 등)

### 2. 전송 실행
- 삼바 전송 화면(/shipments)에서 마켓 계정별로 전송.
- "변경 없는 상품 건너뛰기" 옵션 사용 여부 확인.
- 전송 중 에러 로그를 모니터링한다.

### 3. 전송 후 확인
- 마켓별 성공/실패 건수 집계.
- 실패 건은 원인별로 분류해서 보고:
  - 정책/데이터 문제 (금지어, 카테고리, 필수항목) → `/상품가공` 으로 되돌릴 것
  - 삼바 버그로 의심되는 것 → `/이슈보고` 후보로 표시
  - 마켓 API 일시 오류 → 재시도 대상
