## 판매처별 상세

### 스마트스토어
- **태그 추천 API**: sem=1 + 0.3초 딜레이. 가장 엄격한 제한. 429 시 지수백오프.
- **이미지 업로드**: sem=4. 429 시 재시도.
- **그룹상품 등록**: 0.5초부터 시작하는 점진 백오프. 2분 초과 타임아웃.
- **공식 API 한도**: 네이버 커머스 API — 초당 요청 수 제한 있음 (명시적 문서 미확인)

### 롯데ON
- **등록 후 안정화 대기**: 5초 (`_POST_REGISTER_DELAY`). 등록 직후 API 호출 시 9000/9999 에러 발생하므로 필수.
- **주문 조회**: sem=5. API가 1일 초과 조회를 허용하지 않아 날짜별 병렬 조회.

### G마켓/옥션 (ESMPlus)
- **이미지 업로드 후**: 3초 대기. 등록 직후 API 호출 불가 (2~3분 대기 후 호출 가능).
- **카테고리 조회**: delay=0.5초 (매개변수).
- **API 한도**: 분당 30회. 초과 시 차단.

### Cafe24
- **Rate Limit 모니터링**: X-Api-Call-Limit, x-ratelimit-remaining 헤더로 실시간 추적.
- **429 발생 시**: 2초 대기 후 재시도.
- **옵션 등록 후**: 1초 대기 (variant 생성 대기).

### 쿠팡/11번가
- 현재 코드에서 명시적 속도 제한 없음.
- 쿠팡: HMAC 인증 기반 API, rate limit 정책은 계정별 상이.
- 11번가: 공식 API, 문서 기반 한도 확인 필요.
