# 일일운영 루틴

아침에 `/일일운영` 한 번으로 하루를 시작한다. 아래 순서대로 진행하고, 각 단계 결과를 한 표로 모아 보고한다.

## 순서

1. **오토튠 점검** → `/품절정리` 의 1번(워룸 상태 점검)만 실행
2. **품절 정리** → 밤사이 품절된 상품 확인, 삭제 대상 보고 → 확인 후 삭제
3. **신규 수집** → 오늘 수집 계획이 있으면 `/상품수집` 실행
4. **가공** → 어제 수집분 중 미가공 상품 `/상품가공` 실행
5. **등록** → 가공 완료분 `/상품등록` 실행 (전송 전 확인 필수)
6. **이슈 점검** → 진행 중 발견된 에러 중 이슈보고 후보 정리

## 마감 보고 형식

| 단계 | 결과 |
|---|---|
| 오토튠 | 정상 N대 / 이상 N대 |
| 품절 정리 | 삭제 N건 |
| 수집 | 신규 N건 |
| 가공 | 완료 N건 / 보류 N건 |
| 등록 | 성공 N건 / 실패 N건 |
| 이슈 후보 | N건 (요약) |
